# Python with AI (Exceptions)

## Student Name
Ryan Turner

## Description
This repository contains Python exercises for DATA 4000 focused on using exception handling. The goal is to practice handling errors such as invalid inputs and division issues using try/except blocks.

## Intended Users
Students learning Python and how to handle errors in programs.

## Files Included
- sales_data.py – Handles invalid sales input and calculates revenue  
- inventory_checker.py – Checks inventory levels with error handling  
- age_verification.py – Verifies customer age with input validation  
- financial_ratio.py – Calculates profit margin with error handling  
- salary_adjustment.py – Simulates salary changes with validation
